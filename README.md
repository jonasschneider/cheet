# Cheet

Generate questionable time sheets from Git commit logs.

    $ ./cheet /Trash/eclipse_workspace mail@jonasschneider.com
    2015-02-15: 09:00 -- 14:00
    2015-02-16: 19:30 -- 21:30
    2015-02-18: 19:30 -- 23:30
    2015-02-20: 08:30 -- 14:30
